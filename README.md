# Study Buddy - AI-Powered Learning Platform

Study Buddy is a FREE, open-source alternative to StudyFetch that rivals premium study platforms with AI-powered features.

## Features

- Home Dashboard with study statistics
- My Sets for organizing study materials
- Interactive Calendar with scheduling
- AI Chat Tutor powered by OpenAI
- Flashcards with spaced repetition
- AI-generated Tests & Quizzes
- Notes & Materials with rich text editor
- Audio Recap with text-to-speech
- Essay Grading with AI feedback
- Explainers with step-by-step breakdowns
- Memorizer with active recall
- Day Planner with time blocking
- Competition Prep for AMC 10 and Science Olympiad
- Arcade Games for gamified learning
- Apple-inspired glassmorphism design

## Tech Stack

- React 18 + Vite + TypeScript
- Tailwind CSS + Framer Motion
- Supabase (Backend)
- OpenAI API
- Deployed on Vercel

## Getting Started

```bash
git clone https://github.com/the-X-alien/study-buddy-app.git
cd study-buddy-app
npm install
npm run dev
```
